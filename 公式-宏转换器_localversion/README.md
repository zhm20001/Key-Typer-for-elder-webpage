# KeyMacro 公式宏生成器

将 Markdown/LaTeX 公式转换为网页按键宏脚本。

## 安装步骤

### 1. 安装 Node.js

如果你的电脑还没有安装 Node.js，请按以下步骤操作：

1. 打开浏览器，访问：https://nodejs.org/
2. 点击 **LTS** 版本的下载按钮（推荐）
3. 下载完成后，运行安装程序
4. **全部选择默认选项**，一路点击「Next」或「Install」
5. 安装完成后，关闭安装程序

> 💡 验证是否安装成功：按 `Win + R`，输入 `cmd`，回车，在命令行输入 `node --version`，如果显示版本号（如 `v18.x.x`）说明安装成功。

### 2. 启动工具

1. 双击 **`启动工具.bat`**
2. 会弹出一个黑色窗口，显示「服务器已启动」
3. **不要关闭这个窗口！**

### 3. 开始使用

1. 打开浏览器（Chrome、Edge 等）
2. 地址栏输入：**http://localhost:3000**
3. 回车，打开工具页面

## 使用方法

1. **粘贴公式**：在左侧文本框粘贴你的 Markdown/LaTeX 源码
2. **格式清洗**：点击「本地格式清洗」去除公式符号
3. **复制脚本**：右侧会自动生成脚本，点击「复制代码到剪贴板」
4. **运行脚本**：
   - 在目标网页按 `F12` 打开开发者工具
   - 切换到「Console」标签
   - 粘贴脚本，回车执行
   - **3 秒内**点击网页上的输入框

## 停止工具

使用完毕后，在启动工具的黑色窗口中按 `Ctrl + C`，然后关闭窗口即可。

## 常见问题

**Q: 打开 http://localhost:3000 提示无法访问？**
A: 确保 `启动工具.bat` 正在运行，且窗口未关闭。

**Q: 复制脚本时报错？**
A: 检查是否允许浏览器访问剪贴板，尝试重新点击复制。

**Q: 脚本执行后没有反应？**
A: 确保在倒计时结束前点击了目标输入框，且输入框可以获得焦点。
